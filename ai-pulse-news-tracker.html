AI Pulse - News Monitoring Dashboard

Tableau de bord Streamlit pour suivre l'actualité IA francophone, scorer le sentiment avec Azure AI Language, stocker les articles dans Cosmos DB et interroger les tendances via un assistant RAG.

Lien GitHub Voir les captures

Ce qui a été réalisé

AI Pulse agrège des articles d'actualité autour de l'intelligence artificielle, puis les transforme en indicateurs exploitables : volume d'articles, sentiment moyen, sources les plus actives, sujets dominants et importance des entreprises mentionnées.

Azure AI Language est utilisé pour extraire le sentiment et structurer l'analyse linguistique. Les données sont persistées dans Azure Cosmos DB afin de conserver un historique consultable, filtrable et réutilisable dans les visualisations.

Le tableau de bord Streamlit expose les filtres, graphiques et tables d'exploration, tandis que l'assistant RAG permet de questionner les tendances, les sources, les entreprises, les signaux faibles et les articles importants directement depuis l'application.

Stack technique

Python Streamlit Azure AI Language Azure Cosmos DB RAG NLP Dashboards News monitoring
