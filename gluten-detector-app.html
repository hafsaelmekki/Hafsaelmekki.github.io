Glutify - Détecteur de gluten

Application Streamlit et API FastAPI pour aider les personnes concernées par la maladie cœliaque à analyser des produits alimentaires, comprendre les risques liés au gluten et trouver des alternatives.

Démo Lien GitHub

Ce qui a été réalisé

Glutify centralise l’analyse d’un produit alimentaire dans une interface Streamlit : recherche par nom ou code-barres, récupération des données OpenFoodFacts, affichage de la photo, composition, informations nutritionnelles et verdict lisible.

Le cœur IA combine un LLM Groq avec un moteur RAG. Les règles médicales liées au gluten sont indexées avec FAISS et des embeddings HuggingFace, puis injectées dans le prompt pour produire une décision plus contextualisée : Sans gluten, Risque ou Interdit.

Le projet inclut aussi un backend FastAPI pour exposer les fonctionnalités en REST, un historique auditable des analyses, une recherche d’alternatives sans gluten, des modes recettes pour créer ou adapter des plats, ainsi qu’une gestion de profils et favoris.

Stack technique

Python Streamlit FastAPI RAG FAISS Groq OpenFoodFacts Docker

Fonctionnalités principales

Scan & analyse produit

Recherche par nom ou code-barres, récupération des ingrédients et génération d’un verdict explicite.

RAG médical

Indexation des règles gluten avec FAISS et injection du contexte médical dans l’analyse LLM.

Verdict traçable

Historique des analyses avec date, justification, image produit et badge de risque coloré.

Recettes sans gluten

Création de nouvelles recettes et adaptation de recettes non compatibles vers des alternatives sûres.

API FastAPI

Endpoints REST pour les produits, analyses, recettes, historiques, favoris, utilisateurs et authentification.

Persistance

Sauvegarde des profils, favoris, recettes et analyses avec une base SQLite ou PostgreSQL selon la configuration.

Démo

Le dépôt ne fournit pas encore d’URL de démo publique. La démo peut être lancée en local avec le backend FastAPI et l’interface Streamlit.

Backend

python -m uvicorn api:app --reload --port 8000

Interface

streamlit run app.py
